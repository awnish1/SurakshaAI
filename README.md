# SurakshaAI

## Overview
SurakshaAI is a risk-based identity verification and fraud prevention platform for banking systems.

The platform continuously analyzes:
- User behavior
- Transaction patterns
- Device information
- Login location
- Access history

to generate a dynamic risk score and detect suspicious activities.

## Key Features

### Fraud Detection
- Transaction anomaly detection
- Behavioral analytics
- Risk scoring

### Identity Verification
- OTP verification
- Face verification
- Blink detection

### Insider Threat Detection
- Employee activity monitoring
- Privileged access misuse detection

### Adaptive Authentication
- Low Risk → Allow
- Medium Risk → OTP
- High Risk → OTP + Face Verification
- Critical Risk → Block & Alert

### AI Security Copilot
- Explain fraud alerts
- Assist security analysts
- Generate investigation reports
